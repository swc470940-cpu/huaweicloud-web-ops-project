# 华为云中小企业高可用 Web 站点全栈运维项目

基于华为云从零搭建一套中小企业 Web 站点运维环境，覆盖网络规划、业务部署、高可用架构、监控告警、日志审计、文档沉淀与全流程验证。

## 项目职责

- 独立负责架构设计与落地
- 完成云资源配置、服务部署、可用性验证、运维文档输出

## 技术栈与服务

| 分类 | 服务/技术 |
| --- | --- |
| 云网络 | VPC、安全组、EIP |
| 计算 | ECS |
| 存储 | OBS |
| 数据库 | RDS MySQL |
| 负载均衡 | ELB |
| 弹性伸缩 | AS |
| 监控告警 | SMN、云监控 |
| 审计日志 | CTS |
| 应用服务 | Nginx、WordPress |
| 容器化 | Docker |
| 脚本/自动化 | Shell |
| 文档 | 运维手册、排障记录 |

## 架构说明

- 按业务分层规划 VPC 与子网，通过安全组最小化暴露端口
- ECS 部署 Nginx 与 WordPress，静态资源托管到 OBS
- RDS MySQL 采用主备高可用部署，开启自动备份与内网访问
- 通过 ELB 对外提供访问入口，后端服务器组绑定 ECS
- AS 配置伸缩策略，按业务负载自动扩缩容
- SMN 负责告警通知，CTS 记录关键操作与异常事件
- 编写自动备份脚本、巡检脚本与运维文档，沉淀可复用交付物

## 核心结果

- 完成站点端到端部署，ELB 公网 IP 可正常访问业务
- 实现负载分发、弹性伸缩、告警通知、日志审计闭环可用
- 整理运维手册、操作记录、排障文档，便于后续复现与交接

## 验证结果

- 网站初始化与浏览器预览正常
- 内网访问 RDS 正常，数据库自动备份策略已启用
- 负载均衡、伸缩组、告警主题、订阅者、追踪器配置完成
- 全资源 CTS 审计已开启，日志留存 90 天
- 计费中心收支明细可导出查看
